The _<a href="https://bmi-calculator.aljoudi.ca/" title="BMI Calculator">BMI Calculator</a>_ is a simple Flutter application designed to help users calculate their Body Mass Index (BMI) based on their height and weight.
</div>

<br>
<div style="width: 100%;">
    <a href="#"><img src="https://raw.githubusercontent.com/ferasaljoudi/AssetsRepository/main/BrownSVGs/overview.svg" alt="Overview" style="width: 100%"></a>
</div>
<br>

This app, developed for my ENSE375 class, provides a user-friendly interface where users can input their height and weight to calculate their BMI instantly. It uses a standard BMI formula to determine whether the user's weight is in a healthy range. The app is lightweight and efficient, making it a useful tool for anyone looking to quickly assess their BMI on the go.
